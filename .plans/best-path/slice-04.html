Slice 4 — Messereignis to gültige Runde

P0Delivery #4Foundation

Implement the one automatic timing path and enforce Mindestrundenzeit once in RaceEngine.

Best path forward

  1. SensorTriggered records the authoritative lane and timestamp as MeasurementCaptured.
  2. Treat OfficialStart as each lane’s initial timing reference. Accept the first and later ValidLap only after Mindestrundenzeit from that lane’s previous accepted reference; otherwise append MeasurementRejected with one explicit reason.
  3. Project lap count, last lap, and best lap from accepted events.

Verification

  • cargo test mindestrundenzeit
  • A first trigger before Mindestrundenzeit from OfficialStart is rejected; the first valid trigger and later valid triggers update count and best lap

  • Sector timing
