Slice 24 — Elo after completed races

P3Delivery #25Optional administration

Defer until real race history proves a ranking is useful; derive it deterministically from corrected race results.

Best path forward

  1. Use fixed starting Elo 1500 and K-factor 32. For a multi-lane race, average each driver’s pairwise actual score (win 1, tie 0.5, loss 0) and expected score before applying the standard Elo delta.
  2. Compute current ratings and per-race deltas by folding completed race results in finish order with deterministic rounding; do not store mutable current ratings yet.
  3. A Rundenkorrektur automatically changes that race and every later derived rating. Add a cache only if measured requests exceed 100 ms.
